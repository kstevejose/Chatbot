# Troubleshooting

Common issues and how to fix them.

## 401 Unauthorized
**Cause**: Missing or invalid API key.  
**Fix**: Check the `Authorization` header in your request.

## 403 Forbidden
**Cause**: Your API key does not have access to this resource.  
**Fix**: Check API permissions in the dashboard.

## 500 Internal Server Error
**Cause**: Server issue.  
**Fix**: Try again later or contact support.

## Timeout Issues
Check your internet connection or reduce payload size if making large requests.
